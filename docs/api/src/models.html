src.models

Data models for DShield MCP Elastic SIEM integration.

This module provides comprehensive data models for the DShield MCP server, optimized for DShield SIEM data structures and patterns. It includes models for security events, attacks, reputation data, and various DShield-specific data types.

The models are built using Pydantic for automatic validation, serialization, and documentation generation. They provide type safety and ensure data consistency across the application.

This module provides:

  • Security event models with validation
  • DShield-specific data models
  • Threat intelligence models
  • Query and filter models
  • Statistics and summary models

Example:

from src.models import SecurityEvent, EventSeverity event = SecurityEvent( ... id="event_123", ... timestamp=datetime.now(), ... event_type="attack", ... severity=EventSeverity.HIGH, ... description="Suspicious activity detected" ... ) print(event.severity) EventSeverity.HIGH

 86class SecurityEvent(BaseModel):
 87    """Model for security events from DShield SIEM."""
 88    
 89    id: str = Field(..., description="Unique event identifier")
 90    timestamp: datetime = Field(..., description="Event timestamp")
 91    source_ip: Optional[str] = Field(None, description="Source IP address")
 92    destination_ip: Optional[str] = Field(None, description="Destination IP address")
 93    source_port: Optional[int] = Field(None, description="Source port")
 94    destination_port: Optional[int] = Field(None, description="Destination port")
 95    protocol: Optional[str] = Field(None, description="Network protocol")
 96    event_type: str = Field(..., description="Type of security event")
 97    severity: EventSeverity = Field(EventSeverity.MEDIUM, description="Event severity")
 98    category: EventCategory = Field(EventCategory.OTHER, description="Event category")
 99    description: str = Field(..., description="Event description")
100    
101    # DShield-specific fields
102    country: Optional[str] = Field(None, description="Country of origin")
103    asn: Optional[str] = Field(None, description="Autonomous System Number")
104    organization: Optional[str] = Field(None, description="Organization name")
105    reputation_score: Optional[float] = Field(None, description="DShield reputation score")
106    attack_count: Optional[int] = Field(None, description="Number of attacks from this source")
107    first_seen: Optional[datetime] = Field(None, description="First seen timestamp")
108    last_seen: Optional[datetime] = Field(None, description="Last seen timestamp")
109    tags: List[str] = Field(default_factory=list, description="DShield tags")
110    attack_types: List[str] = Field(default_factory=list, description="Types of attacks")
111    
112    # Metadata
113    raw_data: Dict[str, Any] = Field(default_factory=dict, description="Raw event data")
114    indices: List[str] = Field(default_factory=list, description="Source indices")
115    
116    @field_validator('source_ip', 'destination_ip')
117    @classmethod
118    def validate_ip_address(cls, v: Optional[str]) -> Optional[str]:
119        """Validate IP address format.
120        
121        Args:
122            v: IP address string to validate
123            
124        Returns:
125            The validated IP address string
126            
127        Raises:
128            ValueError: If the IP address format is invalid
129        """
130        if v is not None:
131            try:
132                ipaddress.ip_address(v)
133            except ValueError:
134                raise ValueError(f"Invalid IP address: {v}")
135        return v
136    
137    @field_validator('source_port', 'destination_port')
138    @classmethod
139    def validate_port(cls, v: Optional[int]) -> Optional[int]:
140        """Validate port number.
141        
142        Args:
143            v: Port number to validate
144            
145        Returns:
146            The validated port number
147            
148        Raises:
149            ValueError: If the port number is outside valid range (1-65535)
150        """
151        if v is not None and (v < 1 or v > 65535):
152            raise ValueError(f"Invalid port number: {v}")
153        return v
154    
155    @field_validator('reputation_score')
156    @classmethod
157    def validate_reputation_score(cls, v: Optional[float]) -> Optional[float]:
158        """Validate reputation score range.
159        
160        Args:
161            v: Reputation score to validate
162            
163        Returns:
164            The validated reputation score
165            
166        Raises:
167            ValueError: If the reputation score is outside valid range (0-100)
168        """
169        if v is not None and (v < 0 or v > 100):
170            raise ValueError(f"Reputation score must be between 0 and 100: {v}")
171        return v
