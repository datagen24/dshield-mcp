src.statistical_analysis_tools

Statistical analysis and anomaly detection tools for DShield MCP.

This module provides data-driven anomaly detection utilities backed by Elasticsearch aggregations. It supports multiple detection methods (Z-score, IQR, Isolation Forest, and time-series analysis) and exposes flexible, schema-based aggregation building with optional percentiles. It can generate context pivots (drill-down query templates), calculate calibrated risk scores, and optionally fetch small raw samples for analyst validation.

The module is optimized to avoid heavy queries by default ("fast" modes) while allowing more robust methods (true IQR, MAD, rolling z-score, seasonality) via config and function arguments.

Example:

tools = StatisticalAnalysisTools() results = await tools.detect_statistical_anomalies( ... time_range_hours=24, ... anomaly_methods=["iqr", "time_series"], ... time_series_mode="robust", ... enable_iqr=True, ... enable_percentiles=True, ... ) results["success"] True

 241    async def _get_anomaly_aggregations(
 242        self,
 243        time_range_hours: int,
 244        dimensions: list[str],
 245        methods: list[str],
 246        sensitivity: float,
 247        *,
 248        dimension_schema: dict[str, Any] | None = None,
 249        enable_percentiles: bool | None = False,
 250    ) -> dict[str, Any]:
 251        """Build and execute ES aggregations required for detection.
 252
 253        Args:
 254            time_range_hours (int): Time window to query (hours).
 255            dimensions (list[str]): Legacy dimension list (ignored if
 256                ``dimension_schema`` is provided).
 257            methods (list[str]): Requested methods (may influence which aggs to build).
 258            sensitivity (float): Sensitivity parameter (for potential future tuning).
 259            dimension_schema (dict[str, Any] | None): Schema mapping custom dimension names to
 260                aggregation specifications. See ``detect_statistical_anomalies`` for details.
 261            enable_percentiles (bool | None): Whether to add ES ``percentiles`` aggregations
 262                for numeric fields.
 263
 264        Returns:
 265            dict[str, Any]: The ES ``aggregations`` section from the search response.
 266
 267        Raises:
 268            RuntimeError: If the ES search fails.
 269        """
 270        # Build aggregation query (schema-driven if provided)
 271        aggs: dict[str, Any] = {}
 272
 273        def add_percentiles(target_aggs: dict[str, Any], key_name: str, field: str) -> None:
 274            if enable_percentiles:
 275                percs = self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99])
 276                target_aggs[f"{key_name}_percentiles"] = {
 277                    "percentiles": {
 278                        "field": field,
 279                        "percents": percs,
 280                    }
 281                }
 282
 283        if dimension_schema:
 284            for name, spec in dimension_schema.items():
 285                agg_type = str(spec.get("agg", "terms"))
 286                field = str(spec.get("field", name))
 287                size = int(spec.get("size", 100))
 288                interval = spec.get("interval", "1h")
 289                if agg_type == "terms":
 290                    aggs[f"{name}_counts"] = {"terms": {"field": field, "size": size}}
 291                elif agg_type == "stats":
 292                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 293                    add_percentiles(aggs, name, field)
 294                elif agg_type == "percentiles":
 295                    aggs[f"{name}_percentiles"] = {
 296                        "percentiles": {
 297                            "field": field,
 298                            "percents": spec.get(
 299                                "percents",
 300                                self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99]),
 301                            ),
 302                        }
 303                    }
 304                elif agg_type == "histogram":
 305                    aggs[f"{name}_histogram"] = {
 306                        "histogram": {"field": field, "interval": spec.get("bucket", 10)}
 307                    }
 308                elif agg_type == "date_histogram":
 309                    aggs[f"{name}_time_series"] = {
 310                        "date_histogram": {"field": field, "calendar_interval": interval}
 311                    }
 312                else:
 313                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 314                    add_percentiles(aggs, name, field)
 315        else:
 316            for dimension in dimensions:
 317                if dimension in ["source_ip", "destination_ip"]:
 318                    aggs[f"{dimension}_counts"] = {
 319                        "terms": {"field": f"{dimension}.keyword", "size": 1000},
 320                    }
 321                elif dimension in ["destination_port", "bytes_transferred"]:
 322                    aggs[f"{dimension}_stats"] = {
 323                        "stats": {"field": dimension},
 324                    }
 325                    add_percentiles(aggs, dimension, dimension)
 326                elif dimension == "event_rate":
 327                    aggs[f"{dimension}_time_series"] = {
 328                        "date_histogram": {
 329                            "field": "@timestamp",
 330                            "calendar_interval": "1h",
 331                        },
 332                    }
 333
 334        # Execute aggregation query using existing client method
 335        query_body = {
 336            "size": 0,
 337            "query": {
 338                "range": {
 339                    "@timestamp": {
 340                        "gte": f"now-{time_range_hours}h",
 341                        "lte": "now",
 342                    },
 343                },
 344            },
 345            "aggs": aggs,
 346        }
 347
 348        try:
 349            result = await self.es_client.client.search(
 350                index=await self.es_client.get_available_indices(),
 351                body=query_body,
 352            )
 353
 354            return result["aggregations"]
 355
 356        except Exception as e:
 357            logger.error("Failed to get anomaly aggregations", error=str(e))
 358            raise RuntimeError(f"Failed to retrieve aggregation data: {e!s}") from e

 529    async def _apply_iqr_analysis(
 530        self,
 531        anomaly_data: dict[str, Any],
 532        sensitivity: float,
 533        max_anomalies: int,
 534    ) -> dict[str, Any]:
 535        """Apply IQR-based outlier detection using percentiles when available.
 536
 537        Args:
 538            anomaly_data (dict[str, Any]): Aggregated ES data containing ``*_percentiles``
 539                and/or ``*_stats`` blocks.
 540            sensitivity (float): Multiplier for IQR whiskers (e.g., 1.5 => Tukey fences).
 541            max_anomalies (int): Maximum anomalies to include (not heavily used here).
 542
 543        Returns:
 544            dict[str, Any]: IQR analysis summary with per-field bounds or fallback ranges.
 545        """
 546        try:
 547            # Import scientific libraries for analysis
 548            import numpy as np  # noqa: F401
 549
 550            iqr_results = {
 551                "count": 0,
 552                "anomalies": [],
 553                "method": "iqr",
 554                "sensitivity": sensitivity,
 555            }
 556
 557            # Prefer percentiles (true IQR) if present; fallback to range
 558            for field_name, field_data in anomaly_data.items():
 559                base_field = field_name.replace("_percentiles", "").replace("_stats", "")
 560                if field_name.endswith("_percentiles") and "values" in field_data:
 561                    values = field_data["values"]
 562                    q1 = float(values.get("25.0", values.get("25", 0)))
 563                    q3 = float(values.get("75.0", values.get("75", 0)))
 564                    med = float(values.get("50.0", values.get("50", 0)))
 565                    iqr = max(0.0, q3 - q1)
 566                    if iqr > 0:
 567                        lower = q1 - sensitivity * iqr
 568                        upper = q3 + sensitivity * iqr
 569                        iqr_results["anomalies"].append(
 570                            {
 571                                "field": base_field,
 572                                "q1": q1,
 573                                "q3": q3,
 574                                "median": med,
 575                                "iqr": iqr,
 576                                "lower_bound": lower,
 577                                "upper_bound": upper,
 578                            }
 579                        )
 580                elif field_name.endswith("_stats") and "stats" in field_data:
 581                    stats_data = field_data["stats"]
 582                    if "count" in stats_data and stats_data["count"] > 0:
 583                        mean = float(stats_data.get("avg", 0))
 584                        min_val = float(stats_data.get("min", 0))
 585                        max_val = float(stats_data.get("max", 0))
 586                        range_val = max_val - min_val
 587                        if range_val > 0:
 588                            iqr_results["anomalies"].append(
 589                                {
 590                                    "field": base_field,
 591                                    "mean": mean,
 592                                    "range": range_val,
 593                                    "min": min_val,
 594                                    "max": max_val,
 595                                }
 596                            )
 597
 598            iqr_results["count"] = len(iqr_results["anomalies"])
 599            return iqr_results
 600
 601        except ImportError as e:
 602            logger.warning("Required libraries not available for IQR analysis", error=str(e))
 603            return {
 604                "count": 0,
 605                "anomalies": [],
 606                "method": "iqr",
 607                "error": "Required libraries (numpy) not available",

 767        self,
 768        volumes: list[int | float],
 769        timestamps: list[str | None],
 770        sensitivity: float,
 771        seasonality_hour_of_day: bool,
 772    ) -> list[dict[str, Any]]:
 773        """Compute robust time-series anomalies using median/MAD and rolling z-score.
 774
 775        Falls back to fast mode if NumPy is unavailable.
 776
 777        Args:
 778            volumes: Sequence of counts per bucket.
 779            timestamps: ISO-like timestamps per bucket (or None).
 780            sensitivity: Threshold for robust_z / rolling_z.
 781            seasonality_hour_of_day: Whether to adjust baseline by hour-of-day.
 782
 783        Returns:
 784            List of anomaly entries with robust metrics.
 785        """
 786        try:
 787            import numpy as np  # type: ignore
 788        except Exception:
 789            # Fallback to fast mode
 790            return self._time_series_anomalies_fast(volumes, timestamps, sensitivity)
 791
 792        anomalies: list[dict[str, Any]] = []
 793        if not volumes:
 794            return anomalies
 795        arr = np.array(volumes, dtype=float)
 796        med = float(np.median(arr))
 797        mad = float(np.median(np.abs(arr - med))) or 1.0
 798        window = min(7, max(3, len(arr) // 6))
 799
 800        for i, v in enumerate(arr):
 801            baseline = med
 802            if seasonality_hour_of_day and timestamps[i]:
 803                ts = timestamps[i]
 804                hour = None
 805                if isinstance(ts, str) and len(ts) >= 13:
 806                    try:
 807                        hour = int(ts[11:13])
 808                    except Exception:
 809                        hour = None
 810                if hour is not None:
 811                    hour_vals = []
 812                    for j, t in enumerate(timestamps):
 813                        if isinstance(t, str) and len(t) >= 13:
 814                            try:
 815                                if int(t[11:13]) == hour:
 816                                    hour_vals.append(arr[j])
 817                            except Exception:
 818                                pass
 819                    if hour_vals:
 820                        baseline = float(np.median(np.array(hour_vals)))
 821
 822            robust_z = 0.6745 * (v - baseline) / mad if mad != 0 else 0.0
 823            if i >= window:
 824                recent = arr[i - window : i]
 825                mu = float(np.mean(recent))
 826                sd = float(np.std(recent)) or 1.0
 827                rolling_z = (v - mu) / sd
 828            else:
 829                rolling_z = robust_z
 830
 831            if abs(robust_z) > sensitivity or abs(rolling_z) > sensitivity:
 832                anomalies.append(
 833                    {
 834                        "timestamp": timestamps[i] or f"bucket_{i}",
 835                        "volume": float(v),
 836                        "median": med,
 837                        "mad": mad,
 838                        "robust_z": robust_z,
 839                        "rolling_z": rolling_z,
 840                        "anomaly_threshold": sensitivity,
 841                    }
 842                )

 269            RuntimeError: If the ES search fails.

 271        # Build aggregation query (schema-driven if provided)
 272        aggs: dict[str, Any] = {}
 273
 274        def add_percentiles(target_aggs: dict[str, Any], key_name: str, field: str) -> None:
 275            if enable_percentiles:
 276                percs = self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99])
 277                target_aggs[f"{key_name}_percentiles"] = {

 279                        "field": field,

 290                if agg_type == "terms":
 291                    aggs[f"{name}_counts"] = {"terms": {"field": field, "size": size}}
 292                elif agg_type == "stats":
 293                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 294                    add_percentiles(aggs, name, field)
 295                elif agg_type == "percentiles":
 296                    aggs[f"{name}_percentiles"] = {
 297                        "percentiles": {
 298                            "field": field,
 299                            "percents": spec.get(
 300                                "percents",
 301                                self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99]),
 302                            ),
 303                        }
 304                    }
 305                elif agg_type == "histogram":
 306                    aggs[f"{name}_histogram"] = {
 307                        "histogram": {"field": field, "interval": spec.get("bucket", 10)}
 308                    }
 309                elif agg_type == "date_histogram":
 310                    aggs[f"{name}_time_series"] = {
 311                        "date_histogram": {"field": field, "calendar_interval": interval}
 312                    }
 313                else:
 314                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 315                    add_percentiles(aggs, name, field)
 316        else:
 317            for dimension in dimensions:
 318                if dimension in ["source_ip", "destination_ip"]:
 319                    aggs[f"{dimension}_counts"] = {
 320                        "terms": {"field": f"{dimension}.keyword", "size": 1000},
 321                    }
 322                elif dimension in ["destination_port", "bytes_transferred"]:
 323                    aggs[f"{dimension}_stats"] = {
 324                        "stats": {"field": dimension},
 325                    }
 326                    add_percentiles(aggs, dimension, dimension)
 327                elif dimension == "event_rate":
 328                    aggs[f"{dimension}_time_series"] = {
 329                        "date_histogram": {
 330                            "field": "@timestamp",
 331                            "calendar_interval": "1h",
 332                        },
 333                    }
 334
 335        # Execute aggregation query using existing client method
 336        query_body = {
 337            "size": 0,
 338            "query": {
 339                "range": {
 340                    "@timestamp": {
 341                        "gte": f"now-{time_range_hours}h",
 342                        "lte": "now",
 343                    },
 344                },
 345            },
 346            "aggs": aggs,
 347        }
 348
 349        try:
 350            result = await self.es_client.client.search(
 351                index=await self.es_client.get_available_indices(),
 352                body=query_body,
 353            )
 354
 355            return result["aggregations"]
 356
 357        except Exception as e:
 358            logger.error("Failed to get anomaly aggregations", error=str(e))
 359            raise RuntimeError(f"Failed to retrieve aggregation data: {e!s}") from e

 532        anomaly_data: dict[str, Any],

 538        Args:
 539            anomaly_data (dict[str, Any]): Aggregated ES data containing ``*_percentiles``

 550
 551            iqr_results = {
 552                "count": 0,
 553                "anomalies": [],

 557
 558            # Prefer percentiles (true IQR) if present; fallback to range
 559            for field_name, field_data in anomaly_data.items():
 560                base_field = field_name.replace("_percentiles", "").replace("_stats", "")

 619    async def _apply_isolation_forest_analysis(
 620        self,
 621        anomaly_data: dict[str, Any],
 622        sensitivity: float,
 623        max_anomalies: int,
 624        *,
 625        min_samples: int = 20,
 626        scale_features: bool = True,
 627    ) -> dict[str, Any]:
 628        """Apply Isolation Forest to multivariate feature summaries.
 629
 630        Args:
 631            anomaly_data (dict[str, Any]): Aggregated ES data with ``*_stats`` for fields.
 632            sensitivity (float): Scales the contamination parameter (sensitivity/10 capped at 0.1).
 633            max_anomalies (int): Max anomalies to include.
 634            min_samples (int): Minimum rows required to fit a model; returns a warning otherwise.
 635            scale_features (bool): Whether to mean/standardize features for stability.
 636
 637        Returns:
 638            dict[str, Any]: Isolation Forest analysis including anomaly scores and features.
 639        """
 640        try:
 641            import numpy as np
 642            from sklearn.ensemble import IsolationForest
 643
 644            isolation_results = {
 645                "count": 0,
 646                "anomalies": [],
 647                "method": "isolation_forest",
 648                "sensitivity": sensitivity,
 649            }
 650
 651            # For Isolation Forest, we need to prepare features from the aggregated data
 652            # This is a simplified implementation - in practice, we'd need more data
 653            features = []
 654            feature_names = []
 655
 656            for field_name, field_data in anomaly_data.items():
 657                if field_name.endswith("_stats") and "stats" in field_data:
 658                    stats_data = field_data["stats"]
 659                    if "count" in stats_data and stats_data["count"] > 0:
 660                        features.append(
 661                            [
 662                                stats_data.get("avg", 0),
 663                                stats_data.get("min", 0),
 664                                stats_data.get("max", 0),
 665                                stats_data.get("std_deviation", 0),
 666                            ]
 667                        )
 668                        feature_names.append(field_name.replace("_stats", ""))
 669
 670            if features:
 671                # Convert to numpy array
 672                X = np.array(features)
 673
 674                # Guardrail: minimum sample rows to avoid unstable models
 675                if X.shape[0] < max(1, int(min_samples)):
 676                    return {
 677                        "count": 0,
 678                        "anomalies": [],
 679                        "method": "isolation_forest",
 680                        "warning": f"insufficient samples: {X.shape[0]} < {min_samples}",
 681                    }
 682
 683                # Optional feature scaling for stability
 684                if scale_features:
 685                    means = X.mean(axis=0)
 686                    stds = X.std(axis=0)
 687                    stds[stds == 0] = 1.0
 688                    X = (X - means) / stds
 689
 690                # Apply Isolation Forest

 692                    contamination=min(0.1, sensitivity / 10),  # Scale sensitivity to contamination

 694                )
 695
 696                # Fit and predict
 697                predictions = iso_forest.fit_predict(X)
 698                anomaly_indices = np.where(predictions == -1)[0]
 699
 700                isolation_results["count"] = len(anomaly_indices)
 701                isolation_results["anomalies"] = [
 702                    {
 703                        "field": feature_names[i],
 704                        "anomaly_score": float(iso_forest.score_samples([X[i]])[0]),
 705                        "features": features[i].tolist(),
 706                    }
 707                    for i in anomaly_indices[:max_anomalies]
 708                ]
 709
 710            return isolation_results
 711
 712        except ImportError as e:
 713            logger.warning(
 714                "Required libraries not available for Isolation Forest analysis", error=str(e)
 715            )
 716            return {
 717                "count": 0,
 718                "anomalies": [],
 719                "method": "isolation_forest",
 720                "error": "Required libraries (scikit-learn, numpy) not available",
 721            }
 722        except Exception as e:
 723            logger.error("Isolation Forest analysis failed", error=str(e))
 724            return {
 725                "count": 0,
 726                "anomalies": [],
 727                "method": "isolation_forest",
 728                "error": f"Analysis failed: {e!s}",
 729            }

 737        """Compute simple mean-based deviations for a time series.
 738
 739        Args:
 740            volumes: Sequence of counts per bucket.
 741            timestamps: ISO-like timestamps per bucket (or None).
 742            sensitivity: Threshold scaling factor (mapped to mean-deviation).
 743
 744        Returns:
 745            List of anomaly entries with deviation metrics.
 746        """
 747        anomalies: list[dict[str, Any]] = []
 748        if not volumes:
 749            return anomalies
 750        avg_volume = sum(volumes) / len(volumes)
 751        if avg_volume <= 0:
 752            return anomalies
 753        for i, volume in enumerate(volumes):
 754            deviation = abs(volume - avg_volume) / avg_volume
 755            if deviation > (sensitivity / 10):
 756                anomalies.append(
 757                    {
 758                        "timestamp": timestamps[i] or f"bucket_{i}",
 759                        "volume": volume,
 760                        "average_volume": avg_volume,
 761                        "deviation": deviation,
 762                        "anomaly_threshold": sensitivity / 10,
 763                    }
 764                )

 767    def _time_series_anomalies_robust(
 768        self,
 769        volumes: list[int | float],
 770        timestamps: list[str | None],
 771        sensitivity: float,
 772        seasonality_hour_of_day: bool,
 773    ) -> list[dict[str, Any]]:
 774        """Compute robust time-series anomalies using median/MAD and rolling z-score.
 775
 776        Falls back to fast mode if NumPy is unavailable.
 777
 778        Args:
 779            volumes: Sequence of counts per bucket.
 780            timestamps: ISO-like timestamps per bucket (or None).
 781            sensitivity: Threshold for robust_z / rolling_z.
 782            seasonality_hour_of_day: Whether to adjust baseline by hour-of-day.
 783
 784        Returns:
 785            List of anomaly entries with robust metrics.
 786        """
 787        try:
 788            import numpy as np  # type: ignore
 789        except Exception:
 790            # Fallback to fast mode
 791            return self._time_series_anomalies_fast(volumes, timestamps, sensitivity)
 792
 793        anomalies: list[dict[str, Any]] = []
 794        if not volumes:
 795            return anomalies
 796        arr = np.array(volumes, dtype=float)
 797        med = float(np.median(arr))
 798        mad = float(np.median(np.abs(arr - med))) or 1.0
 799        window = min(7, max(3, len(arr) // 6))
 800
 801        for i, v in enumerate(arr):
 802            baseline = med
 803            if seasonality_hour_of_day and timestamps[i]:
 804                ts = timestamps[i]
 805                hour = None
 806                if isinstance(ts, str) and len(ts) >= 13:
 807                    try:
 808                        hour = int(ts[11:13])
 809                    except Exception:
 810                        hour = None
 811                if hour is not None:
 812                    hour_vals = []
 813                    for j, t in enumerate(timestamps):
 814                        if isinstance(t, str) and len(t) >= 13:
 815                            try:
 816                                if int(t[11:13]) == hour:
 817                                    hour_vals.append(arr[j])
 818                            except Exception:
 819                                pass
 820                    if hour_vals:
 821                        baseline = float(np.median(np.array(hour_vals)))
 822
 823            robust_z = 0.6745 * (v - baseline) / mad if mad != 0 else 0.0
 824            if i >= window:
 825                recent = arr[i - window : i]
 826                mu = float(np.mean(recent))
 827                sd = float(np.std(recent)) or 1.0
 828                rolling_z = (v - mu) / sd
 829            else:
 830                rolling_z = robust_z
 831
 832            if abs(robust_z) > sensitivity or abs(rolling_z) > sensitivity:
 833                anomalies.append(
 834                    {
 835                        "timestamp": timestamps[i] or f"bucket_{i}",
 836                        "volume": float(v),
 837                        "median": med,
 838                        "mad": mad,
 839                        "robust_z": robust_z,
 840                        "rolling_z": rolling_z,
 841                        "anomaly_threshold": sensitivity,
 842                    }
 843                )
 844        return anomalies

Detect anomalies across dimensions using configurable methods.

This function builds aggregations (schema-driven if provided), executes them against Elasticsearch, applies selected anomaly detection methods, and returns a concise analysis (optionally with drill-down templates and raw sample snippets).

Args: time_range_hours (int): Time window in hours to analyze. anomaly_methods (list[str] | None): Methods to apply. Supported: zscore, iqr, isolation_forest, time_series. If None, defaults to ["zscore", "iqr"]. sensitivity (float): Sensitivity parameter used by methods (e.g., z-score multiplier, IQR multiplier, thresholds). Default is 2.5. dimensions (list[str] | None): Legacy dimension list for quick setup (e.g., ["source_ip", "destination_port", ...]). Ignored when dimension_schema is provided. Default includes common fields. return_summary_only (bool): Reserved for future use; currently no effect. max_anomalies (int): Max anomalies to include per method. dimension_schema (dict[str, Any] | None): Data-driven schema mapping dimension names to aggregation specs with keys: field, agg (terms, stats, percentiles, histogram, date_histogram), and optional size, interval, percents. enable_iqr (bool | None): Whether to enable true IQR analysis.
