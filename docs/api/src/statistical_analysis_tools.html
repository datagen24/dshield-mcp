src.statistical_analysis_tools

Statistical analysis and anomaly detection tools for DShield MCP.

This module provides data-driven anomaly detection utilities backed by Elasticsearch aggregations. It supports multiple detection methods (Z-score, IQR, Isolation Forest, and time-series analysis) and exposes flexible, schema-based aggregation building with optional percentiles. It can generate context pivots (drill-down query templates), calculate calibrated risk scores, and optionally fetch small raw samples for analyst validation.

The module is optimized to avoid heavy queries by default ("fast" modes) while allowing more robust methods (true IQR, MAD, rolling z-score, seasonality) via config and function arguments.

Example:

tools = StatisticalAnalysisTools() results = await tools.detect_statistical_anomalies( ... time_range_hours=24, ... anomaly_methods=["iqr", "time_series"], ... time_series_mode="robust", ... enable_iqr=True, ... enable_percentiles=True, ... ) results["success"] True

 239    async def _get_anomaly_aggregations(
 240        self,
 241        time_range_hours: int,
 242        dimensions: list[str],
 243        methods: list[str],
 244        sensitivity: float,
 245        *,
 246        dimension_schema: dict[str, Any] | None = None,
 247        enable_percentiles: bool | None = False,
 248    ) -> dict[str, Any]:
 249        """Build and execute ES aggregations required for detection.
 250
 251        Args:
 252            time_range_hours (int): Time window to query (hours).
 253            dimensions (list[str]): Legacy dimension list (ignored if
 254                ``dimension_schema`` is provided).
 255            methods (list[str]): Requested methods (may influence which aggs to build).
 256            sensitivity (float): Sensitivity parameter (for potential future tuning).
 257            dimension_schema (dict[str, Any] | None): Schema mapping custom dimension names to
 258                aggregation specifications. See ``detect_statistical_anomalies`` for details.
 259            enable_percentiles (bool | None): Whether to add ES ``percentiles`` aggregations
 260                for numeric fields.
 261
 262        Returns:
 263            dict[str, Any]: The ES ``aggregations`` section from the search response.
 264
 265        Raises:
 266            RuntimeError: If the ES search fails.
 267        """
 268        # Build aggregation query (schema-driven if provided)
 269        aggs: dict[str, Any] = {}
 270
 271        def add_percentiles(target_aggs: dict[str, Any], key_name: str, field: str) -> None:
 272            if enable_percentiles:
 273                percs = self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99])
 274                target_aggs[f"{key_name}_percentiles"] = {
 275                    "percentiles": {
 276                        "field": field,
 277                        "percents": percs,
 278                    }
 279                }
 280
 281        if dimension_schema:
 282            for name, spec in dimension_schema.items():
 283                agg_type = str(spec.get("agg", "terms"))
 284                field = str(spec.get("field", name))
 285                size = int(spec.get("size", 100))
 286                interval = spec.get("interval", "1h")
 287                if agg_type == "terms":
 288                    aggs[f"{name}_counts"] = {"terms": {"field": field, "size": size}}
 289                elif agg_type == "stats":
 290                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 291                    add_percentiles(aggs, name, field)
 292                elif agg_type == "percentiles":
 293                    aggs[f"{name}_percentiles"] = {
 294                        "percentiles": {
 295                            "field": field,
 296                            "percents": spec.get(
 297                                "percents",
 298                                self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99]),
 299                            ),
 300                        }
 301                    }
 302                elif agg_type == "histogram":
 303                    aggs[f"{name}_histogram"] = {
 304                        "histogram": {"field": field, "interval": spec.get("bucket", 10)}
 305                    }
 306                elif agg_type == "date_histogram":
 307                    aggs[f"{name}_time_series"] = {
 308                        "date_histogram": {"field": field, "calendar_interval": interval}
 309                    }
 310                else:
 311                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 312                    add_percentiles(aggs, name, field)
 313        else:
 314            for dimension in dimensions:
 315                if dimension in ["source_ip", "destination_ip"]:
 316                    aggs[f"{dimension}_counts"] = {
 317                        "terms": {"field": f"{dimension}.keyword", "size": 1000},
 318                    }
 319                elif dimension in ["destination_port", "bytes_transferred"]:
 320                    aggs[f"{dimension}_stats"] = {
 321                        "stats": {"field": dimension},
 322                    }
 323                    add_percentiles(aggs, dimension, dimension)
 324                elif dimension == "event_rate":
 325                    aggs[f"{dimension}_time_series"] = {
 326                        "date_histogram": {
 327                            "field": "@timestamp",
 328                            "calendar_interval": "1h",
 329                        },
 330                    }
 331
 332        # Execute aggregation query using existing client method
 333        query_body = {
 334            "size": 0,
 335            "query": {
 336                "range": {
 337                    "@timestamp": {
 338                        "gte": f"now-{time_range_hours}h",
 339                        "lte": "now",
 340                    },
 341                },
 342            },
 343            "aggs": aggs,
 344        }
 345
 346        try:
 347            result = await self.es_client.client.search(
 348                index=await self.es_client.get_available_indices(),
 349                body=query_body,
 350            )
 351
 352            return result["aggregations"]
 353
 354        except Exception as e:
 355            logger.error("Failed to get anomaly aggregations", error=str(e))
 356            raise RuntimeError(f"Failed to retrieve aggregation data: {e!s}") from e

 527    async def _apply_iqr_analysis(
 528        self,
 529        anomaly_data: dict[str, Any],
 530        sensitivity: float,
 531        max_anomalies: int,
 532    ) -> dict[str, Any]:
 533        """Apply IQR-based outlier detection using percentiles when available.
 534
 535        Args:
 536            anomaly_data (dict[str, Any]): Aggregated ES data containing ``*_percentiles``
 537                and/or ``*_stats`` blocks.
 538            sensitivity (float): Multiplier for IQR whiskers (e.g., 1.5 => Tukey fences).
 539            max_anomalies (int): Maximum anomalies to include (not heavily used here).
 540
 541        Returns:
 542            dict[str, Any]: IQR analysis summary with per-field bounds or fallback ranges.
 543        """
 544        try:
 545            # Import scientific libraries for analysis
 546            import numpy as np  # noqa: F401
 547
 548            iqr_results = {
 549                "count": 0,
 550                "anomalies": [],
 551                "method": "iqr",
 552                "sensitivity": sensitivity,
 553            }
 554
 555            # Prefer percentiles (true IQR) if present; fallback to range
 556            for field_name, field_data in anomaly_data.items():
 557                base_field = field_name.replace("_percentiles", "").replace("_stats", "")
 558                if field_name.endswith("_percentiles") and "values" in field_data:
 559                    values = field_data["values"]
 560                    q1 = float(values.get("25.0", values.get("25", 0)))
 561                    q3 = float(values.get("75.0", values.get("75", 0)))
 562                    med = float(values.get("50.0", values.get("50", 0)))
 563                    iqr = max(0.0, q3 - q1)
 564                    if iqr > 0:
 565                        lower = q1 - sensitivity * iqr
 566                        upper = q3 + sensitivity * iqr
 567                        iqr_results["anomalies"].append(
 568                            {
 569                                "field": base_field,
 570                                "q1": q1,
 571                                "q3": q3,
 572                                "median": med,
 573                                "iqr": iqr,
 574                                "lower_bound": lower,
 575                                "upper_bound": upper,
 576                            }
 577                        )
 578                elif field_name.endswith("_stats") and "stats" in field_data:
 579                    stats_data = field_data["stats"]
 580                    if "count" in stats_data and stats_data["count"] > 0:
 581                        mean = float(stats_data.get("avg", 0))
 582                        min_val = float(stats_data.get("min", 0))
 583                        max_val = float(stats_data.get("max", 0))
 584                        range_val = max_val - min_val
 585                        if range_val > 0:
 586                            iqr_results["anomalies"].append(
 587                                {
 588                                    "field": base_field,
 589                                    "mean": mean,
 590                                    "range": range_val,
 591                                    "min": min_val,
 592                                    "max": max_val,
 593                                }
 594                            )
 595
 596            iqr_results["count"] = len(iqr_results["anomalies"])
 597            return iqr_results
 598
 599        except ImportError as e:
 600            logger.warning("Required libraries not available for IQR analysis", error=str(e))
 601            return {
 602                "count": 0,
 603                "anomalies": [],
 604                "method": "iqr",
 605                "error": "Required libraries (numpy) not available",
 606            }
 607        except Exception as e:

 764    def _time_series_anomalies_robust(
 765        self,
 766        volumes: list[int | float],
 767        timestamps: list[str | None],
 768        sensitivity: float,
 769        seasonality_hour_of_day: bool,
 770    ) -> list[dict[str, Any]]:
 771        """Compute robust time-series anomalies using median/MAD and rolling z-score.
 772
 773        Falls back to fast mode if NumPy is unavailable.
 774
 775        Args:
 776            volumes: Sequence of counts per bucket.
 777            timestamps: ISO-like timestamps per bucket (or None).
 778            sensitivity: Threshold for robust_z / rolling_z.
 779            seasonality_hour_of_day: Whether to adjust baseline by hour-of-day.
 780
 781        Returns:
 782            List of anomaly entries with robust metrics.
 783        """
 784        try:
 785            import numpy as np  # type: ignore
 786        except Exception:
 787            # Fallback to fast mode
 788            return self._time_series_anomalies_fast(volumes, timestamps, sensitivity)
 789
 790        anomalies: list[dict[str, Any]] = []
 791        if not volumes:
 792            return anomalies
 793        arr = np.array(volumes, dtype=float)
 794        med = float(np.median(arr))
 795        mad = float(np.median(np.abs(arr - med))) or 1.0
 796        window = min(7, max(3, len(arr) // 6))
 797
 798        for i, v in enumerate(arr):
 799            baseline = med
 800            if seasonality_hour_of_day and timestamps[i]:
 801                ts = timestamps[i]
 802                hour = None
 803                if isinstance(ts, str) and len(ts) >= 13:
 804                    try:
 805                        hour = int(ts[11:13])
 806                    except Exception:
 807                        hour = None
 808                if hour is not None:
 809                    hour_vals = []
 810                    for j, t in enumerate(timestamps):
 811                        if isinstance(t, str) and len(t) >= 13:
 812                            try:
 813                                if int(t[11:13]) == hour:
 814                                    hour_vals.append(arr[j])
 815                            except Exception:
 816                                pass
 817                    if hour_vals:
 818                        baseline = float(np.median(np.array(hour_vals)))
 819
 820            robust_z = 0.6745 * (v - baseline) / mad if mad != 0 else 0.0
 821            if i >= window:
 822                recent = arr[i - window : i]
 823                mu = float(np.mean(recent))
 824                sd = float(np.std(recent)) or 1.0
 825                rolling_z = (v - mu) / sd
 826            else:
 827                rolling_z = robust_z
 828
 829            if abs(robust_z) > sensitivity or abs(rolling_z) > sensitivity:
 830                anomalies.append(
 831                    {
 832                        "timestamp": timestamps[i] or f"bucket_{i}",
 833                        "volume": float(v),
 834                        "median": med,
 835                        "mad": mad,
 836                        "robust_z": robust_z,
 837                        "rolling_z": rolling_z,
 838                        "anomaly_threshold": sensitivity,
 839                    }
 840                )

 266        Raises:
 267            RuntimeError: If the ES search fails.
 268        """
 269        # Build aggregation query (schema-driven if provided)
 270        aggs: dict[str, Any] = {}
 271
 272        def add_percentiles(target_aggs: dict[str, Any], key_name: str, field: str) -> None:
 273            if enable_percentiles:
 274                percs = self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99])
 275                target_aggs[f"{key_name}_percentiles"] = {
 276                    "percentiles": {
 277                        "field": field,
 278                        "percents": percs,

 285                field = str(spec.get("field", name))
 286                size = int(spec.get("size", 100))
 287                interval = spec.get("interval", "1h")
 288                if agg_type == "terms":
 289                    aggs[f"{name}_counts"] = {"terms": {"field": field, "size": size}}
 290                elif agg_type == "stats":
 291                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 292                    add_percentiles(aggs, name, field)
 293                elif agg_type == "percentiles":
 294                    aggs[f"{name}_percentiles"] = {
 295                        "percentiles": {
 296                            "field": field,
 297                            "percents": spec.get(
 298                                "percents",
 299                                self.analysis_cfg.get("percentiles", [25, 50, 75, 95, 99]),
 300                            ),
 301                        }
 302                    }
 303                elif agg_type == "histogram":
 304                    aggs[f"{name}_histogram"] = {
 305                        "histogram": {"field": field, "interval": spec.get("bucket", 10)}
 306                    }
 307                elif agg_type == "date_histogram":
 308                    aggs[f"{name}_time_series"] = {
 309                        "date_histogram": {"field": field, "calendar_interval": interval}
 310                    }
 311                else:
 312                    aggs[f"{name}_stats"] = {"stats": {"field": field}}
 313                    add_percentiles(aggs, name, field)
 314        else:
 315            for dimension in dimensions:
 316                if dimension in ["source_ip", "destination_ip"]:
 317                    aggs[f"{dimension}_counts"] = {
 318                        "terms": {"field": f"{dimension}.keyword", "size": 1000},
 319                    }
 320                elif dimension in ["destination_port", "bytes_transferred"]:
 321                    aggs[f"{dimension}_stats"] = {
 322                        "stats": {"field": dimension},
 323                    }
 324                    add_percentiles(aggs, dimension, dimension)
 325                elif dimension == "event_rate":
 326                    aggs[f"{dimension}_time_series"] = {
 327                        "date_histogram": {
 328                            "field": "@timestamp",
 329                            "calendar_interval": "1h",
 330                        },
 331                    }
 332
 333        # Execute aggregation query using existing client method
 334        query_body = {
 335            "size": 0,
 336            "query": {
 337                "range": {
 338                    "@timestamp": {
 339                        "gte": f"now-{time_range_hours}h",
 340                        "lte": "now",
 341                    },
 342                },
 343            },
 344            "aggs": aggs,
 345        }
 346
 347        try:
 348            result = await self.es_client.client.search(
 349                index=await self.es_client.get_available_indices(),
 350                body=query_body,
 351            )
 352
 353            return result["aggregations"]
 354
 355        except Exception as e:
 356            logger.error("Failed to get anomaly aggregations", error=str(e))
 357            raise RuntimeError(f"Failed to retrieve aggregation data: {e!s}") from e

 529        self,
 530        anomaly_data: dict[str, Any],

 534        """Apply IQR-based outlier detection using percentiles when available.

 536        Args:
 537            anomaly_data (dict[str, Any]): Aggregated ES data containing ``*_percentiles``

 548

 550                "count": 0,
 551                "anomalies": [],

 553                "sensitivity": sensitivity,

 555
 556            # Prefer percentiles (true IQR) if present; fallback to range
 557            for field_name, field_data in anomaly_data.items():
 558                base_field = field_name.replace("_percentiles", "").replace("_stats", "")

 617    async def _apply_isolation_forest_analysis(
 618        self,
 619        anomaly_data: dict[str, Any],
 620        sensitivity: float,
 621        max_anomalies: int,
 622        *,
 623        min_samples: int = 20,
 624        scale_features: bool = True,
 625    ) -> dict[str, Any]:
 626        """Apply Isolation Forest to multivariate feature summaries.
 627
 628        Args:
 629            anomaly_data (dict[str, Any]): Aggregated ES data with ``*_stats`` for fields.
 630            sensitivity (float): Scales the contamination parameter (sensitivity/10 capped at 0.1).
 631            max_anomalies (int): Max anomalies to include.
 632            min_samples (int): Minimum rows required to fit a model; returns a warning otherwise.
 633            scale_features (bool): Whether to mean/standardize features for stability.
 634
 635        Returns:
 636            dict[str, Any]: Isolation Forest analysis including anomaly scores and features.
 637        """
 638        try:
 639            import numpy as np
 640            from sklearn.ensemble import IsolationForest
 641
 642            isolation_results = {
 643                "count": 0,
 644                "anomalies": [],
 645                "method": "isolation_forest",
 646                "sensitivity": sensitivity,
 647            }
 648
 649            # For Isolation Forest, we need to prepare features from the aggregated data
 650            # This is a simplified implementation - in practice, we'd need more data
 651            features = []
 652            feature_names = []
 653
 654            for field_name, field_data in anomaly_data.items():
 655                if field_name.endswith("_stats") and "stats" in field_data:
 656                    stats_data = field_data["stats"]
 657                    if "count" in stats_data and stats_data["count"] > 0:
 658                        features.append(
 659                            [
 660                                stats_data.get("avg", 0),
 661                                stats_data.get("min", 0),
 662                                stats_data.get("max", 0),
 663                                stats_data.get("std_deviation", 0),
 664                            ]
 665                        )
 666                        feature_names.append(field_name.replace("_stats", ""))
 667
 668            if features:
 669                # Convert to numpy array
 670                X = np.array(features)
 671
 672                # Guardrail: minimum sample rows to avoid unstable models
 673                if X.shape[0] < max(1, int(min_samples)):
 674                    return {
 675                        "count": 0,
 676                        "anomalies": [],
 677                        "method": "isolation_forest",
 678                        "warning": f"insufficient samples: {X.shape[0]} < {min_samples}",
 679                    }
 680
 681                # Optional feature scaling for stability
 682                if scale_features:
 683                    means = X.mean(axis=0)
 684                    stds = X.std(axis=0)
 685                    stds[stds == 0] = 1.0
 686                    X = (X - means) / stds
 687
 688                # Apply Isolation Forest
 689                iso_forest = IsolationForest(
 690                    contamination=min(0.1, sensitivity / 10),  # Scale sensitivity to contamination
 691                    random_state=42,
 692                )
 693
 694                # Fit and predict
 695                predictions = iso_forest.fit_predict(X)
 696                anomaly_indices = np.where(predictions == -1)[0]
 697
 698                isolation_results["count"] = len(anomaly_indices)
 699                isolation_results["anomalies"] = [
 700                    {
 701                        "field": feature_names[i],
 702                        "anomaly_score": float(iso_forest.score_samples([X[i]])[0]),
 703                        "features": features[i].tolist(),
 704                    }
 705                    for i in anomaly_indices[:max_anomalies]
 706                ]
 707
 708            return isolation_results
 709
 710        except ImportError as e:
 711            logger.warning(
 712                "Required libraries not available for Isolation Forest analysis", error=str(e)
 713            )
 714            return {
 715                "count": 0,
 716                "anomalies": [],
 717                "method": "isolation_forest",
 718                "error": "Required libraries (scikit-learn, numpy) not available",
 719            }
 720        except Exception as e:
 721            logger.error("Isolation Forest analysis failed", error=str(e))
 722            return {
 723                "count": 0,
 724                "anomalies": [],
 725                "method": "isolation_forest",
 726                "error": f"Analysis failed: {e!s}",
 727            }

 735        """Compute simple mean-based deviations for a time series.
 736
 737        Args:
 738            volumes: Sequence of counts per bucket.
 739            timestamps: ISO-like timestamps per bucket (or None).
 740            sensitivity: Threshold scaling factor (mapped to mean-deviation).
 741
 742        Returns:
 743            List of anomaly entries with deviation metrics.
 744        """
 745        anomalies: list[dict[str, Any]] = []
 746        if not volumes:
 747            return anomalies
 748        avg_volume = sum(volumes) / len(volumes)
 749        if avg_volume <= 0:
 750            return anomalies
 751        for i, volume in enumerate(volumes):
 752            deviation = abs(volume - avg_volume) / avg_volume
 753            if deviation > (sensitivity / 10):
 754                anomalies.append(
 755                    {
 756                        "timestamp": timestamps[i] or f"bucket_{i}",
 757                        "volume": volume,
 758                        "average_volume": avg_volume,
 759                        "deviation": deviation,
 760                        "anomaly_threshold": sensitivity / 10,
 761                    }
 762                )

 765    def _time_series_anomalies_robust(
 766        self,
 767        volumes: list[int | float],
 768        timestamps: list[str | None],
 769        sensitivity: float,
 770        seasonality_hour_of_day: bool,
 771    ) -> list[dict[str, Any]]:
 772        """Compute robust time-series anomalies using median/MAD and rolling z-score.
 773
 774        Falls back to fast mode if NumPy is unavailable.
 775
 776        Args:
 777            volumes: Sequence of counts per bucket.
 778            timestamps: ISO-like timestamps per bucket (or None).
 779            sensitivity: Threshold for robust_z / rolling_z.
 780            seasonality_hour_of_day: Whether to adjust baseline by hour-of-day.
 781
 782        Returns:
 783            List of anomaly entries with robust metrics.
 784        """
 785        try:
 786            import numpy as np  # type: ignore
 787        except Exception:
 788            # Fallback to fast mode
 789            return self._time_series_anomalies_fast(volumes, timestamps, sensitivity)
 790
 791        anomalies: list[dict[str, Any]] = []
 792        if not volumes:
 793            return anomalies
 794        arr = np.array(volumes, dtype=float)
 795        med = float(np.median(arr))
 796        mad = float(np.median(np.abs(arr - med))) or 1.0
 797        window = min(7, max(3, len(arr) // 6))
 798
 799        for i, v in enumerate(arr):
 800            baseline = med
 801            if seasonality_hour_of_day and timestamps[i]:
 802                ts = timestamps[i]
 803                hour = None
 804                if isinstance(ts, str) and len(ts) >= 13:
 805                    try:
 806                        hour = int(ts[11:13])
 807                    except Exception:
 808                        hour = None
 809                if hour is not None:
 810                    hour_vals = []
 811                    for j, t in enumerate(timestamps):
 812                        if isinstance(t, str) and len(t) >= 13:
 813                            try:
 814                                if int(t[11:13]) == hour:
 815                                    hour_vals.append(arr[j])
 816                            except Exception:
 817                                pass
 818                    if hour_vals:
 819                        baseline = float(np.median(np.array(hour_vals)))
 820
 821            robust_z = 0.6745 * (v - baseline) / mad if mad != 0 else 0.0
 822            if i >= window:
 823                recent = arr[i - window : i]
 824                mu = float(np.mean(recent))
 825                sd = float(np.std(recent)) or 1.0
 826                rolling_z = (v - mu) / sd
 827            else:
 828                rolling_z = robust_z
 829
 830            if abs(robust_z) > sensitivity or abs(rolling_z) > sensitivity:
 831                anomalies.append(
 832                    {
 833                        "timestamp": timestamps[i] or f"bucket_{i}",
 834                        "volume": float(v),
 835                        "median": med,
 836                        "mad": mad,
 837                        "robust_z": robust_z,
 838                        "rolling_z": rolling_z,
 839                        "anomaly_threshold": sensitivity,
 840                    }
 841                )
 842        return anomalies

Detect anomalies across dimensions using configurable methods.

This function builds aggregations (schema-driven if provided), executes them against Elasticsearch, applies selected anomaly detection methods, and returns a concise analysis (optionally with drill-down templates and raw sample snippets).

Args: time_range_hours (int): Time window in hours to analyze. anomaly_methods (list[str] | None): Methods to apply. Supported: zscore, iqr, isolation_forest, time_series. If None, defaults to ["zscore", "iqr"]. sensitivity (float): Sensitivity parameter used by methods (e.g., z-score multiplier, IQR multiplier, thresholds).
